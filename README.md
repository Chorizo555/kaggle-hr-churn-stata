# Kaggle HR Churn Competition (Stata)

## Objective
Predict employee attrition using logistic regression.

## Method
- Logistic regression
- Variable selection via AUC
- Multicollinearity check (VIF)

## Model Evolution
1. Baseline logistic regression
2. Variable refinement
3. Category expansion (job role)

## Result
- Train AUC: 0.775
- Public Score: 0.710

## Tool
Stata 19.5 BE
